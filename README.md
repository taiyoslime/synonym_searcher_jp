## synonym_searcher_jp

All data are fetched from [連想類語辞典 日本語シソーラス](http://renso-ruigo.com/)

## Usage

```
$ synjp 具象

具象的　　　　　　　　　　　　　　具体性　　　　　　　　　　　　　　有相　　　　　　　　　　　　　　　象形
形像　　　　　　　　　　　　　　　モノローグ　　　　　　　　　　　　リフレイン　　　　　　　　　　　　概念くだき
直喩　　　　　　　　　　　　　　　表象　　　　　　　　　　　　　　　実相観入　　　　　　　　　　　　　様式
.....
```
